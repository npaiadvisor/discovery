# Maintenance Assistant — system prompt (template)

> **Stub.** A system prompt for the conversational assistant a nonprofit talks to when it wants to change its system. Loaded into whichever tool the client already uses (a Claude Project, a ChatGPT Custom GPT, a Copilot agent, or a Gemini Gem).

The finished version: grounds the assistant in the client's runbook; teaches it the ask → PR → preview → merge loop; gates any change behind a human-readable preview; and keeps an explicit off-limits list (auth, secrets, cost caps, the schedule).
